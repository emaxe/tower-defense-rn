# Kenney — Tower Defense (Top-Down)

**Ссылка:** https://kenney.nl/assets/tower-defense-top-down

**Лицензия:** Creative Commons CC0 (полная свобода, включая коммерческое использование)

**Характеристики:**
- Формат: 2D top-down (вид сверху)
- Количество файлов: 300+ (PNG/SVG)
- Разрешение: разное, оптимизировано для игр

## Содержимое набора

### Тайлы местности (terrain tiles)
- Зелёная трава — основной газон/поля
- Коричневая земля/грязь — дорожки и тропинки
- Серый камень/бетон/асфальт — укреплённые участки
- Голубая вода/лёд
- Песчаные/бежевые участки — пустыня/пляж
- Переходные тайлы — угловые и краевые варианты для сопряжения поверхностей

### Башни и юниты
- Красная круглая турель с серым основанием (основная пушечная башня)
- Серая ударная башня/молот
- Истребитель/авиационный юнит
- Зелёная боевая машина/танк
- Ракета с огненным следом

### Снаряды и эффекты
- Жёлтые/оранжевые пули и выстрелы
- Ракеты
- Взрывы (предположительно)

### Декор
- Кусты
- Деревья
- Прочие декоративные элементы карты

## Оценка для проекта

### Плюсы
- Top-down стиль идеально ложится на нашу tile-based сетку
- PNG можно вставлять напрямую через `<Image>` в React Native
- Есть и тайлы для фона, и спрайты башен, и снаряды — полный набор
- Из одного пака можно собрать 3 разных уровня (трава, вода/лёд, песок)
- CC0 — можно использовать в любых целях без ограничений

### Минусы
- В превью не видны спрайты врагов (мобов) — возможно, нужно докачивать врагов отдельно (например, из Kenney Monster Pack)
- Стиль довольно простой/кондиционный — не «высокое искусство», но читаемый и функциональный

## Рекомендации по использованию
- Скачать и положить в `assets/kenney/td-top-down/`
- Тайлы использовать для фона уровней
- Башни — для замены SVG-прямоугольников
- Снаряды — для анимации выстрелов
- Для врагов при необходимости докачать Kenney Monster Pack или другой набор

## Альтернативы (если стиль не подойдёт)
- itch.io — Tower Defense Assets (Pixel Mush) — пиксель-арт
- CraftPix — Free Fields Tileset Pixel Art — для фона
- Kenney — Monster Pack — для врагов (если отдельно нужны)
